# SEIP Report Formatting Guide

## General formatting

Use consistent formatting across all pages.

## Page title

```text
Font size: 20–24
Weight: Bold
Position: Top left
```

## KPI cards

```text
Font size: 24–32 for values
Title size: 10–12
Display units: Auto or None
```

## Percentages

Use:

```text
1 decimal place
```

Example:

```text
65.0%
```

## Matrix

Use conditional formatting on:

```text
Unemployment Rate %
Youth Unemployment Rate %
Average Quality Score
```

## Slicers

Recommended slicer style:

```text
Dropdown
Single select: Off
Search: On
```

## Accessibility

- Do not rely only on colour.
- Use clear titles.
- Use descriptive tooltips.
- Keep enough contrast between text and background.
