# SEIP Phase 5A — Power BI Dashboard Page Specifications

## Dashboard 1 — Executive Overview

### Purpose

Give government, NGOs and project sponsors a quick summary of employment conditions.

### KPI cards

- Total Job Seekers
- Unemployed Count
- Unemployment Rate %
- Youth Unemployment Rate %
- Average Months Unemployed
- Training Interest Rate %

### Visuals

| Visual | Fields |
|---|---|
| Filled map / Azure map | Township, Unemployment Rate % |
| Bar chart | Township vs Unemployed Count |
| Donut chart | Employment status split |
| Line chart | Survey date vs Total Survey Responses |
| Matrix | Township, Total Job Seekers, Unemployed Count, Unemployment Rate % |

### Slicers

- Township
- Ward
- Gender
- Age group
- Highest qualification
- Survey date
